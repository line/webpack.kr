HMR로 알려져 있는 [Hot Module Replacement](/concepts/hot-module-replacement)을 활성화합니다.

W> HMR은 **절대** 프로덕션에 사용되서는 안됩니다.

## Basic Usage

HMR을 활성화하는 것은 간단하며 대부분의 경우 옵션이 필요하지 않습니다.

```javascript
new webpack.HotModuleReplacementPlugin({
  // Options...
});
```
