`minChunkSize`보다 작은 청크를 병합하여 청크 크기를 설정한 한계 이상으로 유지합니다.

```js
new webpack.optimize.MinChunkSizePlugin({
  minChunkSize: 10000, // 최소 문자 수
});
```

## Usage via CLI

해당 플러그인과 플러그인 옵션은 [CLI](/api/cli/)를 통해서도 실행할 수 있습니다.

```bash
webpack --optimize-min-chunk-size 10000
```
